## API

| 参数     | 说明         | 类型                      | 默认值         |
| -------- | ------------ | ------------------------- | -------------- | --- |
| type     | 按钮类型     | default/primary/text/link | (确定取消按钮) |
| size     | 按钮大小     | small/middle/large        | middle         |     |
| loading  | 按钮加载状态 | boolean                   | false`         |     |
| icon     | 图标         | ReactNode                 | -              |     |
| disabled | 禁用状态     | boolean                   | -              |     |
